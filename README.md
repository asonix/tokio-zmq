# Tokio ZMQ

This crate contains wrappers around ZeroMQ Concepts with Futures.

Currently Supported Sockets
 - REP
 - REQ
 - PUB
 - SUB
 - PUSH
 - PULL

See the [examples folder](https://github.com/asonix/zmq-futures/tree/master/examples) for usage examples.
